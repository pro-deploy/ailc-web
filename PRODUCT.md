# PRODUCT.md — ailc (AI Life Cycle), лендинг

register: brand

## Что это

ailc (AI Life Cycle) — автономный офлайновый оркестратор качества и безопасности кода,
один защищённый бинарь на Rust со встроенной мультиязычной моделью и пятнадцатью
грамматиками разбора. Подключается в редактор кода по протоколу MCP (Model Context
Protocol). Гарантию «можно сдавать или нельзя» даёт детерминированный гейт, а не нейросеть.

Лендинг — это продающая одностраничная витрина. Его задача: за тридцать секунд довести
посетителя до вставки одной строки в конфигурацию своего редактора, после чего готовый
бинарь скачивается сам и следующий запрос к модели уже проходит через гейт ailc.

## Аудитория

Разработчики, использующие ИИ-агентов в редакторе (Cline и GitHub Copilot в Visual Studio
Code, Claude Code, Cursor) с локальной или облачной моделью. От вайбкодера до
энтерпрайз-джуна. Им нужен ноль возни: ни сборки из исходников, ни возни со скриптами
установки. Добавил строку в конфиг, и всё работает.

## Главный оффер

Одна строка в конфиг редактора. Бинарь скачивается сам, инициализируется в проект,
следующий запрос к нейросети идёт уже через детерминированный гейт качества и безопасности.

## Три канала подключения

1. npx (проще всего, если есть Node.js): `{ "command": "npx", "args": ["-y", "ailc-mcp"] }`.
   Обёртка сама тянет бинарь нужной платформы из релиза на GitHub и запускает `ailc serve`.
2. Docker (без Node.js, контейнер с готовым бинарём): `docker run -i --rm -v <проект>:/work`.
3. Локальная сборка из исходников на GitHub: для тех, кто хочет форкнуть и доработать.

## Ключевые факты для текстов

74 инструмента в 9 семействах. 15 языков глубокого анализа (Python, JavaScript, TypeScript,
Go, Java, Ruby, PHP, C#, Rust, Kotlin, Scala, C, C++, Swift, Dart). Межпроцедурный
анализ заражения (taint), статический анализ безопасности по дереву разбора (SAST), матрица
OWASP Top 10, безопасность ИИ-приложений (OWASP LLM Top 10), комплаенс РФ (152-ФЗ,
242-ФЗ), отчёт SARIF 2.1.0 для непрерывной интеграции, Definition of Done с тремя
состояниями вердикта. Полностью офлайн, без внешних сервисов. Лицензия Apache 2.0 на
открытое ядро. Репозиторий github.com/pro-deploy/ailc.

## Голос бренда

Точный, бронированный, бескомпромиссный. Не игривый. Инструмент-эталон, измерительный
прибор, контрольный штамп. Доверие через строгость и детерминизм.

## Регистр и направление

Тёмная тема (требование заказчика), технологичный строгий стиль. Концепция: приборная панель
контроля, которая выносит вердикт «прошло или нет». Не генерический терминал на чёрном фоне.
